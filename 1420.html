Save desktop
This command saves the current desktop configuration to a desktop.

If no name is specified in the text box, IDA saves the current desktop configuration to the <Database> or <Database debugger> desktop (if the debugger is active) in the database.

If you enter a name, IDA saves the current configuration to a named desktop in the registry.

If you select 'Default', IDA saves the current configuration to the <Default> or <Default debugger> desktop (if the debugger is active) in the registry.
