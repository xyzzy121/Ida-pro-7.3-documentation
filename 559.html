Problem: Too many lines

 
 Description:
        The current item (instruction or data) occupies more lines
        on the screen than it is allowed by the current configuration.

 What to do:
        1. If the current item is an array or String literal, try to divide it.
     or 2. Delete Additional comment lines
     or 3. Disable cross-references display
     or 4. Increase the limit in IDA.CFG, the paramter name is MAX_ITEM_LINES.
