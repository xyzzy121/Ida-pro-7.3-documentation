Append Function Tail

 
 Action    name: AppendFunctionTail
This command appends an arbitrary range of the program to a function definition. A range must be selected before applying this command. This range must not intersect with other function chunks.

IDA will ask to select the parent function for the selection and will append the range to the function definition.
